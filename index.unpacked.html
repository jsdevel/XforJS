X is a Template Language for Javascript. You can serve up compiled or raw templates (compiled is preferred ;).

Documentation

What follows is the documentation for X. This document will try to include the actual integration tests and source files that are used during development of the compiler. Following this practice allows the source code of the compiler to remain in sync with the documentation and vice versa.
Every editor herein is executable, meaning you can change the contents of the textareas and immediately see what the compiler is doing. Go ahead. Change the value of "showRatings" to "false" in the Input Params section.

X for JS

A Template Lang that would make Mama proud.

X is a Templating Language for Javascript. You can serve up precompiled templates and compile them in a web browser, or you can serve them up on the back end with a js enabled server.

Documentation

What follows is the documentation for X. This document will try to include the actual integration tests and source files that are used during development of the compiler. Following this practice allows the compiler to remain nsync with the docs and vice versa.
Every editor herein is executable, meaning you can change the contents of the textareas and immediately see what the compiler is doing.
